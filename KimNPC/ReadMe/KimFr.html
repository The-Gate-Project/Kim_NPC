DESCRITPION :

Personnage recrutable pour Baldur's Gate II, vous pourrez la trouvez à coté de Renald Bloodscalp.

 Incroyablement belle, délicieusement sensuelle... et terriblement chieuse. Kim la pirate fait escale à Athkathla. Saurez-vous l'apprivoiser ?

Kim est romancable par tous les hommes qu'ils soient humains, elfes ou de demi-elfes



INSTALLATION :

Extraire le contenu de l'archive dans votre répertoire de jeu

 Lancer Setup-Kim_NPC.exe

Suivre les indications.
Kim

Procédure d'upgrade d'une version 1.4x ou antérieure vers la version actuelle:

Vous ne pouvez faire cet upgrade qui si Kim n'est pas encore venue dans votre équipe. 
  1. Désintaller la vieille version du module
  2. Supprimer le dossier KimNPC qui se trouve dans votre dossier d'intallation de Baldur's Gate II
  3. Supprimer les 3 fichiers Setup-Kim.exe, Setup-Kim.tp2, SETUP-KIM.DEBUG qui se trouvent aussi dans votre dossier d'intallation de Baldur's Gate II
  4. Dézipper et installer la dernière version du module
